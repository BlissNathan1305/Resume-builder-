# Premium Resume Builder

Create professional resumes online with multiple templates and color themes.

## Features
- Three templates: Classic, Modern, Creative
- Multiple color themes: Blue, Navy, Green, Dark Gray
- Profile photo support
- Live preview in Streamlit
- Download in Word (.docx) and PDF
- Optional sample resume for quick testing

## Deployment
1. Push this repository to GitHub.
2. Go to [Streamlit Cloud](https://share.streamlit.io) and connect the repo.
3. Deploy and get your public URL.
